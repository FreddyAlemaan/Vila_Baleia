# 🐋 Restaurante Vila Baleia - Sitio Web

## 📖 Descripción
Este proyecto es la página web oficial del restaurante familiar **Vila Baleia**, ubicado en Porto Moniz, Madeira. Bajo el lema *"Onde o Mar Encontra a Mesa"*, el sitio web está diseñado para reflejar la esencia del restaurante, ofreciendo a los clientes una experiencia digital elegante e intuitiva. A través de la plataforma, los usuarios pueden explorar la oferta gastronómica, conocer la historia del local (desde 2011) y gestionar sus reservas fácilmente.

## ✨ Secciones del Sitio
El proyecto cuenta con una navegación fluida estructurada en las siguientes secciones principales:
* **Início:** Página principal con un diseño inmersivo, reseñas de clientes y accesos directos a reservas.
* **Menu:** Catálogo digital con nuestra oferta de sabores locales e ingredientes frescos.
* **Sobre Nós:** Nuestra historia, ambiente familiar y conexión con la región.
* **Contacto:** Información de ubicación, horarios y vías de comunicación.
* **Reservar Mesa:** Sistema integrado para solicitar mesas de forma rápida y sencilla.

## 🚀 Tecnologías Utilizadas
Este proyecto fue construido priorizando un rendimiento óptimo y un diseño moderno (UI/UX):
* **Frontend:** React.js
* **Herramienta de Construcción:** Vite
* **Estilos:** Tailwind CSS
